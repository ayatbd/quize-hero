Q.1: What is the defference among map, filter, for each and find
Ans: Difference between map() and forEach() is the returning value. The forEach() method returns undefined and map() returns a new array with the transformed elements. Even if they do the same job, the returning value remains different.
filter() returns an array containing the element that satisfies the condition, but find() returns the element itself that satisfies the condition. In filter() , whole array is iterated despite the fact that the element being searched for is present at the beginning.
Q.2: What is the diffrence between local storage, session storage and cookies
Ans: Local storage is useful for storing data that the user will need to access later, such as offline data. Session storage is a great way to improve the performance of your web applications. Cookies are a good choice for storing data that should not be persisted for a long time, such as session IDs.
